# AI PDF Agent

A FastAPI-based service for AI-powered PDF processing and interaction.

## Project Structure

```
ai-pdf-agent/
├── app/
│   ├── main.py
│   ├── api/
│   └── services/
├── requirements.txt
├── README.md
└── .gitignore
```

## Setup

1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Run the app:
   ```sh
   uvicorn app.main:app --reload
   ```
